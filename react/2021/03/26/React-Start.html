환경구축

OS는 Windows 환경이다.

설치 항목은 아래와 같다.

  • Node.js ( https://nodejs.org/ko/download/ )

  • Yarn ( npm install --global yarn )

    Node.js과 함께 설치되는 npm을 이용해도 되지만, yarn이 npm보다 더 빠르며 효율적인 캐시 시스템과 기타 부가 기능을 제공하므로 설치한다.

  • Visual Stuido Code (유용한 Extension 항목을 아래와 같다.)
    • ESLint : 자바스크립트 문법 및 코드스타일을 검사해주는 도구
    • Reactjs Code Snippets : 제작자 charalampos karypidis. 코드 스니펫 모음
    • Prettier-Code formatter : 코드 스타일 자동 정리
  • Git ( https://git-scm.com/download/ )

create-react-app으로 프로젝트 생성하기

create-react-app은 리액트 프로젝트를 생성할 때 필요한 웹팩, 바벨의 설치 및 설정 과정을 생략하고 바로 간편하게 프로젝트 작업 환경을 구축해주는 도구이다. 후에 설정을 커스터마이징 할 수도 있다.

hello-react라는 리액트 프로젝트를 하나 만들어보자. 명령어는 아래와 같다.

$ yarn create react-app hello-react  # npm init react-app hello-react
yarn create v1.22.10
[1/4] Resolving packages...
[2/4] Fetching packages...
[3/4] Linking dependencies...
[4/4] Building fresh packages...

... (생략) ...

Happy hacking!
Done in 129.58s.

프로젝트 생성이 완료되었다. 프로젝트 디렉터리로 이동하여 리액트 개발 전용 서버를 구동해 보자

$ cd hello-react
$ yarn start  # npm start

터미널이 아래와 같이 뜨고, 3000번 포트로 리액트 로고가 보이는 검정 페이지 하나가 띄워지면 구동이 된것이다.

Compiled successfully!

You can now view hello-react in the browser.     

  Local:            http://localhost:3000        
  On Your Network:  http://192.168.40.169:3000   

Note that the development build is not optimized.
To create a production build, use yarn build.
