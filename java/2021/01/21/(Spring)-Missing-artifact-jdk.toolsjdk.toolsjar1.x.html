(Spring) Missing artifact jdk.tools:jdk.tools:jar:1.x

by kt1201

January 21, 2021
in Java

maven의 pom.xml에서 Missing artifact jdk.tools:jdk.tools:jar:1.x 에러 발생시 설정

  • 원인
    • jre에 JAVA_HOME이 설정되어 tools.jar를 올바르게 가져오지 못함
    • 프로젝트가 잘못된 jdk를 찾음(내 PC에서만 이런 오류가 났음;; 이유는 아직 못찾음)

  • 해결방법

    방법1. eclipse.ini 파일에 vm 위치추가

-vm
C:\Program Files\Java\jdk1.8.0_281\bin\javaw

​ 방법2. 이클립스 Window > Preferences > Java > Installed JREs에서 Installed JREs에 있는 항목에 jdk 경로가 올바르게 되어있는지 확인하고, 틀리게 되어있다면 jdk 경로를 지정

​ 방법3. pom.xml에 의존성 추가

<dependency>
    <groupId>jdk.tools</groupId>
	<artifactId>jdk.tools</artifactId>
	<version>1.8.0_281</version>
	<scope>system</scope>
	<systemPath>${env.JAVA_HOME}/lib/tools.jar</systemPath>
</dependency>

내경우는 방법1,2를 시도해도 해결이 안되서 방법3을 썼다.
