include 사용

iframe 사용

iframe의 src에 html의 실제 경로를 넣으면 chrome에서

“not allowed to load local resource” 라는 에러가 뜬다.

이때는, 톰캣의 server.xml의 Host태그 안에 다음 코드를 추가한다.

<Context path="/filepath" reloadable="true" docBase="C:\Users\user\git\kcgsb\sb-portal-dev\src\main\webapp\WEB-INF\jsp"/>

아래는 JSP 코드

<iframe src="/filepath/analysis/test.html" width=500 height=500></iframe>
(Postgres) CSV파일 DB에 Import (Spring) 일정 시간 지나면 Connection Pool 연결 끊어짐(connection time out error)
