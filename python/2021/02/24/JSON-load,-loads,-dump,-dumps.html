json load, loads, dump, dumps

by kt1201

February 24, 2021
in Python

json.dump, json.dumps

dump 계열은 파이썬의 객체(dict 등)를 다른 형태로 변환한다.

  • dump 함수는 파이썬 객체를 스트림 객체(파일 등)로 변환한다.

  • dumps 함수는 파이썬 객체를 한줄의(직렬화된) json 문자열로 변환한다.

json.load, json.loads

load 계열은 대상을 파싱해서 파이썬의 객체(dict 등)로 변환한다.

  • load 함수는 스트림 객체(파일 등)에서 데이터를 읽어 파이썬 객체(dict 등)로 변환한다.

  • loads 함수는 json 문자열을 파싱해서 파이썬 객체로 변환한다.

예제 코드는 아래와 같다.

import json

## read json file
with open('myobj.json', 'rb') as f:
  root = json.load(f)
  ...

## write json file
with open('myobj2.json', 'wb') as f:
  json.dump(root, f)
  ...
